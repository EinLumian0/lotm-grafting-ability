# LOTM Grafting Ability

A Spigot Plugin for Minecraft 1.20.2 that adds various grafting abilities (space, sun, pressure, gravity, distance) from Lord of the Mysteries.

## Features
- Space grafting
- Sun grafting
- Pressure grafting
- Gravity grafting
- Distance grafting

## Installation
1. Build the plugin using your IDE.
2. Place the generated `.jar` into your server’s `plugins/` folder.
3. Start the server.

OR

Drop the `lotm_grafting_ability.jar` into your servers plugin folder.

## Commands
- `/grafting give <type>` – Unlock a grafting ability.
